# Support

## Getting Help

For setup and runtime problems, check these first:

- [README.md](README.md)
- [docs/test-setup.md](docs/test-setup.md)
- [docs/troubleshooting.md](docs/troubleshooting.md)

## When Opening an Issue

Include:

- what you tried
- relevant logs
- your deployment mode (`standalone`, external Asterisk, monitoring profile, etc.)
- changed environment variables, without secrets

## Security Issues

Do not use public issues for vulnerabilities. See [SECURITY.md](SECURITY.md).
